# XAI-Guided-Error-Analysis-dan-Penanganan-Kelas-Minoritas-pada-Klasifikasi-Kanker-Tulang
Ini adalah proyek akhir untuk memenuhi kelulusan dari mata kuliah Deep Learning Citra Medis oleh Kelompok 10 program studi Sains Data

## Kesimpulan Akhir Penelitian

  Berdasarkan hasil penelitian yang telah dilakukan mengenai klasifikasi multi-kelas kanker tulang menggunakan citra radiografi dengan pendekatan XAI, menunjukkan bahwa model MobileNetV2 mampu digunakan untuk melakukan klasifikasi citra radiografi kanker tulang ke dalam tiga kelas, yaitu Benign, Malignant, dan Normal. Model baseline menghasilkan akurasi sebesar 64% dengan recallhanya sebesar 26%, sehingga model masih mengalami kesulitan dalam mengenali kelas Malignant yang merupakan kelas minoritas. Kemudian, metode XAI hanya dengan Grad-CAM telah mampu memberikan informasi yang cukup untuk menjelaskan proses pengambilan keputusan model. Visualisasi heatmap menunjukkan area yang menjadi perhatian model pada saat melakukan klasifikasi sehingga kelemahan model dapat diidentifikasi secara lebih jelas. Informasi pada XAI berhasil digunakan untuk mengidentifikasi kelemahan pada model, seperti ketidakseimbangan data, perhatian model pada area yang tidak relevan, dan tingginya kelas Malignant.
	Hasil beberapa eksperimen yang dilakukan menunjukkan bahwa strategi berbasis XAI mampu meningkatkan performa model. Kombinasi Hard Sample Mining dan Targeted Augmentation pada eksperimen E4 menghasilkan performa terbaik dengan akurasi sebesar 77.59% dengan recall Malignant meningkat menjadi 34.7%, sehingga pada E4 ini model lebih pintar mendeteksi kelas minoritas dibandingkan model baseline. Dengan demikian, penelitian ini menunjukkan bahwa XAI tidak hanya berfungsi sebagai alat interpretasi model, tetapi juga dapat dimanfaatkan sebagai dasar dalam menentukan strategi optimasi yang lebih terarah untuk meningkatkan performa klasifikasi kanker tulang.
